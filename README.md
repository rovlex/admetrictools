# Ad Metric Tools

Static Astro V1 for admetrictools.com.

## Pages included

- `/`
- `/roas-calculator/`
- `/cpc-calculator/`
- `/ctr-calculator/`
- `/cpm-calculator/`

## Local setup

```bash
npm install
npm run dev
npm run build
```

## Cloudflare Pages

Use these build settings:

- Framework preset: Astro
- Build command: `npm run build`
- Build output directory: `dist`
- Node version: `22.12.0` or newer

The project includes `.nvmrc` and `.node-version` with `22.12.0`.
